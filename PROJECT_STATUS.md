## 🎯 功能特色

### 安全策略
- **白名单模式**（默认）：只允许白名单内的域名
- **黑名单模式**：拦截黑名单内的域名
- **实时验证**：创建短链时即时检查域名权限
- **智能匹配**：支持精确匹配和子域名匹配

### 跳转模式
- **直接跳转**：无中间页面，直接 302 重定向
- **过渡页面**：显示目标信息，自动倒计时跳转
- **手动确认**：需要用户手动点击确认
- **密码保护**：输入密码后才能访问

### 用户体验
- **多语言支持**：中英文实时切换
- **响应式设计**：适配各种设备屏幕
- **动画效果**：流畅的交互动画
- **错误处理**：友好的错误提示

## 🔧 部署说明

### 开发环境
```bash
# 安装依赖
bun install

# 初始化数据库
bun run setup

# 启动开发服务器
bun run dev
```

### 生产环境
```bash
# 构建项目
bun run build

# 启动生产服务器
bun run start
```

### 环境变量
```bash
# 数据库连接
DATABASE_URL="file:./dev.db"

# 应用基础URL
NEXT_PUBLIC_BASE_URL="http://localhost:3000"
```
